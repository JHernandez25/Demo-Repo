Updating index.html

Each parent <section> in the #webslides element is an individual slide.

Update the html between each <section>tag. When copying a slide template, make sure to copy both opening <section>tag and closing </section>tag.

<article id="webslides">
<!-- *** SCC1* SLIDE # * Conduct Research * TOC *** -->
  <section>
    <h1>Conduct Research</h1>
  </section>
<!-- *** SCC1* SLIDE # * Conduct Research - Question ***  -->
  <section class="bg-primary">
    <div class="wrap">
      <h3>Answer the question in your journal.</h3>
    </div>
  </section>
</article>

SCC1 Tool

SCC1 Tool

The SCC1 tool is a rapid development tool that assists in updating the Table of contents (TOC) and the slide comments. As a webslide project keeps expanding, more time is required to update the comments and the TOC. Updating those elements is repetitive and is possible to make an error. This tool handles the repetitive task and saves you time.

Github Colab

Setup

This section explains how to set up your html code so the SCC tool can successfully execute your HTML code.

Step 1:

Add “SCC1” Keyword to all the slide comments.

<!-- *** SCC1* SLIDE 2 * Table of Contents *** -->

SCC1 keyword triggers the script to watch the comment. SCC1 is required for the script. Not entering SCC1 will cause the script to leave out the comment.

Step 2:

Add “TOC” keyword to the slide comments that will be included in the table of contents.

<!-- *** SCC1* SLIDE 5 * Scope * TOC *** -->

TOC Keyword triggers the script to add the comment to the Table of contents. TOC is optional.

Step 3:

Set up the Table of Contents. The table of contents is divided in 2 parts, start and end. For the start make sure you have the “TABLE OF CONTENTS” keyword on the slide comment.

For the end make sure you have the “Table Of Contents END” keyword after the end of the section. Not entering ““Table Of Contents END” will cause the script to override the project.

<!-- *** SCC1 * SLIDE 2 * TABLE OF CONTENTS *** -->
<section>...     
</section>
<!-- *** Table of Contents END *** -->
